Understanding Federal Accounts

Account

The composite account code combined with its descriptive title

Example: 018-45-0243 Federal Direct Student Loan Program Account

  • 018: Agency identifier (Department of Education)
  • 45: Allocation Transfer Agency Code (Office of Federal Student Aid)
  • 0243: Main account code (Federal Direct Student Loan Program)
  • Title: Federal Direct Student Loan Program Account

Account Number

The Treasury Account Symbol (TAS) using Treasury's standard agency codes from the FAST Book

Example: 91-0243

  • 91: Treasury's designation for Department of Education
  • 0243: Federal Direct Student Loan Program account
