Fund Type Crosswalk

This page explains how Department of Homeland Security budget accounts are categorized by fund type and budget enforcement category.

Data Source

Treasury FAST Book (Federal Account Symbols and Titles)

The fund type classifications come from the U.S. Department of the Treasury's official FAST Book, which provides authoritative account information for all federal agencies.

Fund Type to Budget Category Mapping

Note: This mapping follows standard federal budget classifications. Fund types determine how money can be spent and whether Congressional action is required.
Fund Type Budget Category Description DHS Examples
General Funds Discretionary Annual appropriations that require Congressional action each year Most DHS operations, CBP border security, ICE enforcement, TSA screening
Special Funds Mandatory Earmarked revenues from specific sources, spending set by law USCIS immigration fees, CBP customs user fees, TSA security fees
Revolving Funds Mandatory Self-sustaining funds from business-like operations FEMA flood insurance, Coast Guard supply fund, Working capital funds
Trust Funds Mandatory Funds held in trust for specific purposes, set by law Coast Guard oil spill liability, Sport fish restoration, Gift funds
Deposit Funds Other Temporary holdings, not available for agency use Immigration bonds, seized assets pending adjudication

Technical Implementation

The crosswalk process:

  1. Downloads the latest FAST Book Part II from Treasury (Excel format)
  2. Extracts all DHS accounts (agency code 070)
  3. Maps Treasury Account Symbols (TAS) to fund types
  4. Merges fund types with OpenOMB apportionment data
  5. Applies budget category rules based on fund type
Data Freshness: The FAST Book is updated periodically by Treasury. This crosswalk uses the most recent available version. Budget data comes from OpenOMB.org and may have different update cycles.
